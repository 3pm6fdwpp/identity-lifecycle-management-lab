# Identity Lifecycle Management Lab

## Overview

This hands-on IAM homelab project demonstrates identity lifecycle management workflows using Microsoft Entra ID concepts aligned with enterprise Identity & Access Management (IAM) operations.

The lab focuses on practical IAM administration tasks including:

- User provisioning and de-provisioning
- Role-Based Access Control (RBAC)
- Multi-Factor Authentication (MFA)
- Conditional Access
- Identity governance concepts
- Access reviews
- Least privilege access models
- Authentication and access management workflows

---

## Objectives

- Simulate enterprise IAM administration workflows
- Strengthen understanding of identity governance and access lifecycle management
- Practice secure access control concepts aligned with Zero Trust principles
- Develop hands-on familiarity with Microsoft Entra ID administration concepts
- Improve IAM documentation and operational process skills

---

## Lab Scenarios

### 1. User Provisioning & Onboarding

- Created users and department-based groups
- Assigned RBAC permissions based on job responsibilities
- Applied least privilege access principles
- Documented onboarding workflows

### 2. MFA & Conditional Access

- Configured MFA enforcement policies
- Reviewed authentication security concepts
- Implemented Conditional Access policy scenarios
- Simulated access restriction workflows

### 3. Identity Governance

- Reviewed access governance concepts
- Simulated access reviews and entitlement management workflows
- Evaluated role assignment structures
- Documented identity lifecycle processes

### 4. User De-Provisioning & Access Removal

- Simulated account disablement procedures
- Removed role assignments and group access
- Reviewed secure offboarding practices
- Documented de-provisioning workflows
